Cross Realm - Tactical Edition -
『テイルズ オブ エターニア』の作中ミニゲーム「ウィス（WHISS）」をベースに、オンライン対戦向けに再構築・強化したファンメイドのタクティカル・カードゲームです。
ガラスモーフィズムを採用したネオン輝くサイバーパンクなUIで、友人との白熱したリアルタイム対戦を楽しめます。
⚡ クイックスタート
NAME: プレイヤー名を入力（最大10文字）。
ROOM ID: 対戦相手と同じ合言葉を入力。
SYNC: ルームに入場し、全員が揃ったらホスト（一番最初に入った人）が「EXECUTE」を押して開始。
RECONNECT: 万が一通信が切れても、同じ名前とルームIDで入り直せば即座に手札を保持したまま復帰できます。
🌀 基本ルール
手札を最も早く使い切るか、最後まで生き残ったプレイヤーの勝利です。
初期手札: 7枚
属性サイクル: 場の属性に対して「次の属性」または「同じ属性」を出すことができます。
サイクル: 歯車 ➡ 氷河期 ➡ 噴水 ➡ バッテリー ➡ 機械 ➡ 古文書 ➡ (歯車へ)
バースト制限: 手札が 15枚 に達した瞬間に💀脱落（BURST）となります。脱落者の手札はすべて破棄されます。
🃏 カード構成（戦略的固定51枚）
運要素を抑え、カウンティングによる戦略性を高めるため、デッキ枚数を以下の通り固定しています。
メイン属性（各10枚）
歯車 (GEAR): 通常7枚 / 特殊（+2攻撃）3枚
機械 (MACHINE): 通常7枚 / 特殊（リバース）3枚
噴水 (FOUNTAIN): 通常7枚 / 特殊（WILD）3枚
サブ属性（各5枚）
氷河期 (ICEAGE) / バッテリー (BATTERY) / 古文書 (ARCHIVE)
完全ワイルド（各3枚）
惑星 (PLANET) / 廃墟 (RUINS): いつでも出せ、属性を自由に変更可能。
✨ 特殊効果とショートカット
【+2】累積攻撃: 特殊歯車は次のプレイヤーのドロー枚数を加算します。回避できないと一気にバーストの危険があります。
【REV】反転: 特殊機械は手番の進行方向を逆にします。
【WILD】変転: 特殊噴水、惑星、廃墟は属性を自由に変更できます。
氷河期の例外: 「氷河期」は重ね出しができず、サイクルを止める効果があります。
属性の跳躍:
氷河期 ➡ バッテリー
古文書 ➡ 氷河期
💡 タクティカル・ガイド
手札の自動整理: 常に属性順に並ぶため、戦略の構築に集中できます。
モバイル対応: 手札が多い時は左右にスワイプして確認可能です。
ターンの視覚化: 自分の番には画面がネオンに輝き、「YOUR TURN」の巨大な透かしが表示されます。
