Cross Realm - Tactical Edition -
『テイルズ オブ エターニア』の作中ミニゲーム「ウィス（WHISS）」のルールをベースに、オンライン対戦向けに再構築したファンメイドのタクティカル・カードゲームです。
ガラスモーフィズムを採用したモダンなUIで、友人とのリアルタイム対戦を楽しめます。

⚡ クイックスタート
IDENTIFIER: プレイヤー名を入力（例：Reid）

CHANNEL ID: 対戦相手と同じ合言葉を入力（例：CR-ROOM-01）

SYNC START: ルームに入場し、全員が揃ったらホストが「EXECUTE」を押して開始。

🌀 基本ルール
手札を最も早く使い切ったプレイヤーの勝利です。
自分のターンでは、場の属性に対して 「属性サイクルで次の属性」 または 「同じ属性（一部除く）」 のカードを出すことができます。

属性相関（サイクル）
歯車 (GEAR) → 氷河期 (ICEAGE) → 噴水 (FOUNTAIN) → バッテリー (BATTERY) → 機械 (MACHINE) → 古文書 (ARCHIVE) → (歯車へ戻る)

✨ 特殊効果カード
歯車 (GEAR) [+2]: 次のプレイヤーのドロー枚数を +2 します（累積可能）。

機械 (MACHINE) [REV]: 順番が回る方向を反転させます。

噴水 (FOUNTAIN) [WILD]: いつでも出せ、場の属性を好きなものに変更できます。

惑星 (PLANET) / 廃墟 (RUINS): 属性を持たないワイルドカードです。

💡 タクティカル・チップス
氷河期の停滞: 「氷河期」は同じ属性を重ねて出すことができません。

噴水の救済: 「噴水」はサイクルを無視して、「氷河期」の上に直接重ねて出すことができます。

ワイルド・チェンジ: ワイルドカードで「歯車」を指定した場合、ドロー攻撃（+2）は発生しません。

🛠️ 技術構成
フロントエンド: React 18, Babel

通信: Socket.io 4.x (Real-time Communication)

デザイン: Pure CSS (Glassmorphism design)
